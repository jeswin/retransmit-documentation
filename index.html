Retransmit

HTTP and WebSocket API Gateway

Get Started

For Microservices

Retransmit is designed to handle the plumbing between Microservices and let developers focus on writing business logic.

Traffic Control

Load balancing, metering, response aggregation, rate limiting, circuit breakers, authentication, web jobs and more.

WebSockets Support

Retransmit can stream responses to WebSocket clients by polling multiple backend services or by reading from Redis queues.

Analytics and Monitoring

Real-time dashboards to see what's happening across your application stack. Alerts when your servers go south.

Extensible

Write hooks in JavaScript to customize requests and responses flowing through Retransmit. Write plugins to handle custom protocols.

Free and Open Source

Retransmit Core is Free and Open Source. We are actively looking for contributors and testers. Check our GitHub profile.
